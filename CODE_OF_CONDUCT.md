# Contributor Covenant Code of Conduct

This project has adopted the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its Code of Conduct.

Please refer to the full text at the link above for details on expected behavior, unacceptable behavior, enforcement responsibilities, and reporting guidelines.

## Reporting

Instances of behavior that violates the Code of Conduct may be reported to the project maintainers. All complaints will be reviewed and investigated promptly and fairly.

## Attribution

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org), version 2.1.
